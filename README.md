# 🟩 Minecraft Version Tracker  
A simple, clean, and dynamic website that shows every major Minecraft update — now with a **live visit counter** powered by JSONBin.

## ✨ Features
- 📜 Full list of Minecraft updates  
- 🕒 Release dates + descriptions  
- 🎨 Clean UI with splash screen  
- 🔄 Auto-loading update blocks  
- 🌐 Hosted on GitHub Pages  
- 🔢 **Live Visit Counter** (JSONBin database)

## 🔢 Visit Counter (How It Works)
Every time someone visits the site, the counter increases.  
This works globally — ANY visitor increments the number.

Powered by:
- JSONBin.io (remote database)
- JavaScript fetch API
- Secure Master Key authentication
